# Traveling Weather App

### Purpose

This app is to help people plan for a road trip. You can plan out the weather for the major cities you will pass through.

### Features

1. Google Maps API
2. Weather API
3. Calendar

### Technologies

1. Javascript
2. Angular
3. Node.js
4. Mongo
5. Express
6. API
7. Gulp
8. Mocha/Chai
